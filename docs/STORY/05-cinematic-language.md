# Cinematic Language

## Purpose
Establish the visual storytelling grammar for the portfolio. Every camera movement, transition, composition and animation should reinforce the narrative.

## CIN-001 — Camera
- Third-person cinematic follow camera.
- Smooth spring-based movement.
- Never abrupt cuts except at loading.

## CIN-002 — Composition
- Character positioned using the rule of thirds.
- Important landmarks frame portfolio content.

## CIN-003 — Lighting
- Begin at sunrise.
- Progress naturally to golden hour, sunset and night.
- Day/Night toggle allows visitors to experience both moods without breaking the story.

## CIN-004 — Motion
- Character walk cycle evolves from child to adult.
- NPCs perform subtle ambient actions.
- Trees, grass and water respond to wind.

## CIN-005 — Transitions
- Environment changes continuously.
- Avoid fades between sections.
- Use architecture, terrain and lighting to communicate progression.

## CIN-006 — UI
- UI appears only when context is needed.
- Glassmorphism panels should feel lightweight and secondary to the world.

## Acceptance Criteria
The visitor should feel they are watching a carefully directed interactive film while always understanding that this is a professional portfolio website.
