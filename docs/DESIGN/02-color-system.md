# Color System

# BOOK III — DESIGN

## Purpose
Define the complete color language for the portfolio across Day and Night modes. Every environment, UI component and lighting setup should derive from this system.

## DESIGN-020 — Design Principles
- Nature-inspired palette.
- High readability.
- Cinematic warmth.
- WCAG AA minimum contrast.
- Consistent branding.

## Day Palette
- Sky: Soft Azure
- Sunlight: Warm Gold
- Grass: Natural Green
- Wood: Cedar Brown
- Stone: Neutral Gray
- Water: Clear Blue
- Accent: Vermilion

## Night Palette
- Sky: Deep Indigo
- Moonlight: Cool Blue
- Lanterns: Warm Amber
- UI Accent: Cyan
- Shadows: Blue-gray

## Environmental Rules
- Cherry blossoms become more saturated during golden hour.
- Rivers reflect the sky color.
- Fog shifts from warm in the morning to cool at night.
- Glassmorphism panels adapt to theme.

## Accessibility
- Text contrast ratio >= 4.5:1
- Interactive elements remain visible in both themes.

## Acceptance Criteria
The entire portfolio should feel visually cohesive regardless of whether the visitor selects Day or Night mode.
