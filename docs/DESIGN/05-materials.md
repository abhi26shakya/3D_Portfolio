# Materials & Surface System

# BOOK III — DESIGN

## Purpose
Define the visual material language used throughout the world. Materials should reinforce realism, remain optimized for real-time rendering, and support the portfolio's cinematic style.

## MAT-001 Philosophy
- Stylized realism.
- Physically Based Rendering (PBR).
- Consistent roughness and metallic values.
- Avoid overly reflective surfaces.

## Core Material Library
- Wood: homes, bridges, fences.
- Stone: roads, foundations, shrines.
- Grass: fields and hills.
- Water: rivers and ponds.
- Glass: UI panels and observatory.
- Metal: phone booth, railings, lamps.

## MAT-002 Environmental Rules
- Materials should age naturally.
- Moss appears on older stone.
- Wooden structures show subtle wear.
- Wet surfaces become more reflective after rain.

## MAT-003 Performance
- Reuse textures.
- Limit unique materials.
- Prefer texture atlases where appropriate.

## Acceptance Criteria
Every material should contribute to a cohesive visual identity while remaining performant for WebGL rendering.
