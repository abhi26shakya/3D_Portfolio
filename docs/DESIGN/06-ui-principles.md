# UI Principles

# BOOK III — DESIGN

## Purpose
The user interface should complement the cinematic world rather than compete with it. UI elements must appear intentional, minimal and elegant.

## UI-001 Philosophy
- Content first.
- World first, interface second.
- Minimal distractions.
- Consistent interaction patterns.

## UI-002 Layout
- Floating glass panels.
- Responsive grid.
- Maximum content width for readability.
- Smooth section reveal animations.

## UI-003 Components
- Navigation
- Timeline
- Research cards
- Experience cards
- Project gallery
- Skills visualization
- Contact panel
- Resume download

## UI-004 Motion
- Spring-based transitions.
- Fade and translate animations.
- Respect reduced-motion preferences.

## UI-005 Accessibility
- Keyboard navigation.
- Visible focus states.
- WCAG AA contrast.
- Screen-reader friendly structure.

## UI-006 Day/Night
The UI should automatically adapt colors, shadows and glass effects when switching themes.

## Acceptance Criteria
The interface should feel premium, modern and unobtrusive while ensuring visitors can quickly access every section of the portfolio.
