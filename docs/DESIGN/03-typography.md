# Typography System

# BOOK III — DESIGN

## Purpose
Define the complete typography system for the portfolio. Typography should communicate professionalism, clarity and elegance while remaining highly readable across desktop, tablet and mobile.

## TYPO-001 Font Families
Primary: Geist Sans (or Inter fallback)
Monospace: Geist Mono (or JetBrains Mono)

## TYPO-002 Hierarchy
Display XL: Hero title
Display L: Section titles
Heading: Cards
Body: Portfolio content
Caption: Metadata
Code: Technical snippets

## TYPO-003 Rules
- Maximum line length: 75 characters.
- Base font size: 16px.
- Responsive scaling using CSS clamp().
- High contrast in both Day and Night modes.

## TYPO-004 Motion
- Headings fade and slide gently.
- Paragraphs stagger into view.
- No excessive kinetic typography.

## TYPO-005 Accessibility
- WCAG AA compliant.
- Respect browser zoom.
- Maintain readability at 200% scaling.

## Acceptance Criteria
Typography should guide attention without competing with the 3D world. Visitors should be able to comfortably read all portfolio content while remaining immersed in the cinematic experience.
