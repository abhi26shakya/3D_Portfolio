# Character Anatomy

Book VIII — Character

## Overview

Character anatomy defines the physical structure, proportions, and movement capabilities of characters. A strong anatomical foundation ensures believable animations, realistic deformation, and visual consistency across the project.

## Core Anatomy

- Head
- Neck
- Torso
- Arms
- Hands
- Legs
- Feet

## Proportion Guidelines

- Realistic human proportions
- Stylized proportions
- Hero character proportions
- NPC proportions

## Movement Considerations

- Joint placement
- Range of motion
- Balance and posture
- Center of gravity
- Weight distribution

## Modeling Considerations

- Clean topology
- Animation-friendly edge loops
- Symmetry during modeling
- Proper mesh density

## Best Practices

- Keep anatomy consistent across characters.
- Model with animation in mind.
- Validate deformations before rigging.
- Use reference images for accuracy.
- Document anatomical standards.

A well-designed anatomical structure forms the basis for expressive animation, efficient rigging, and high-quality character production.
