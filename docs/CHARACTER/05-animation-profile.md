# Animation Profile

Book VIII — Character

## Overview

An animation profile defines how a character moves, reacts, and expresses emotions within the 3D Portfolio. It serves as a blueprint for animators and developers to maintain consistent behavior across all interactions.

## Animation Categories

- Idle
- Walk
- Run
- Jump
- Interact
- Celebrate
- Wave
- Sit

## Facial Animations

- Smile
- Blink
- Surprise
- Thinking
- Talking

## State Machine

- Idle → Walk
- Walk → Run
- Run → Jump
- Jump → Land
- Idle → Interaction

## Technical Considerations

- Frame rate consistency
- Root motion support
- Animation blending
- Transition timing
- Performance optimization

## Best Practices

- Keep animations readable and responsive.
- Reuse animation clips where possible.
- Blend transitions smoothly.
- Test animations in-game.
- Document animation states and triggers.

A well-defined animation profile ensures fluid movement, believable character behavior, and a polished user experience.
