# 3D Models and Meshes

Book VI — Assets

## Overview

3D models define the geometry of objects in a scene. A mesh consists of vertices, edges, and faces that describe an object's shape.

## Common Model Types

- Static Meshes
- Skeletal Meshes
- Procedural Meshes
- Collision Meshes

## Geometry Concepts

- Vertices
- Edges
- Faces
- Normals
- UV Coordinates
- Topology

## File Formats

- FBX
- glTF
- OBJ
- STL
- USD

## Best Practices

- Keep polygon counts appropriate for the target platform.
- Maintain clean topology.
- Reuse modular assets.
- Apply proper naming conventions.
- Optimize meshes using Levels of Detail (LOD).
