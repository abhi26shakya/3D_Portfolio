# Animations and Rigging

Book VI — Assets

## Overview

Animations bring 3D models to life by defining motion over time, while rigging provides the skeletal structure and control systems required to animate characters, props, and creatures efficiently.

## Core Concepts

- Skeletons and Bones
- Joints
- Skinning and Weight Painting
- Inverse Kinematics (IK)
- Forward Kinematics (FK)
- Blend Shapes / Morph Targets

## Animation Types

- Idle
- Walk
- Run
- Jump
- Interaction
- Cinematic Sequences

## Common Formats

- FBX
- glTF
- BVH
- USD

## Best Practices

- Maintain consistent bone naming.
- Reuse animation clips when possible.
- Optimize keyframes and compression.
- Validate rigs before export.
- Organize animation assets by character and state.

A robust animation and rigging pipeline enables realistic movement, efficient reuse, and smooth integration into game engines and rendering systems.
