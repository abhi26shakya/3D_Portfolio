# Materials and Textures

Book VI — Assets

## Overview

Materials define how surfaces interact with light, while textures provide visual detail such as color, roughness, metallic properties, normals, and ambient occlusion. Together they create realistic and stylized appearances in 3D scenes.

## Texture Maps

- Albedo (Base Color)
- Normal Map
- Roughness Map
- Metallic Map
- Ambient Occlusion (AO)
- Emissive Map
- Height / Displacement Map

## Material Workflows

- Physically Based Rendering (PBR)
- Metallic-Roughness Workflow
- Specular-Glossiness Workflow

## Best Practices

- Use consistent texture resolutions.
- Compress textures appropriately.
- Reuse shared materials.
- Optimize GPU memory usage.
- Follow naming conventions for assets.

High-quality materials and textures significantly enhance the realism and visual consistency of a project.
