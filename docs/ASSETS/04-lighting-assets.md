# Lighting Assets

Book VI — Assets

## Overview

Lighting assets define how scenes are illuminated and greatly influence realism, mood, and visual storytelling. They include light sources, environment lighting, probes, and supporting configuration data.

## Types of Lights

- Directional Lights
- Point Lights
- Spot Lights
- Area Lights
- Ambient Lighting
- HDRI Environment Maps

## Supporting Assets

- Reflection Probes
- Light Probes
- Shadow Maps
- Skyboxes

## Optimization

- Bake static lighting when possible.
- Limit dynamic lights.
- Use cascaded shadows selectively.
- Optimize probe placement.
- Balance visual quality with performance.

Well-designed lighting assets enhance immersion while maintaining real-time rendering performance.
