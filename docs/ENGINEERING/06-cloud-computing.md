# Cloud Computing

Book V — Engineering

## Overview

Cloud computing enables on-demand access to computing resources such as servers, storage, databases, networking, and AI services over the internet. It allows engineering teams to build scalable, resilient, and cost-efficient systems.

## Service Models

- Infrastructure as a Service (IaaS)
- Platform as a Service (PaaS)
- Software as a Service (SaaS)
- Function as a Service (FaaS)

## Deployment Models

- Public Cloud
- Private Cloud
- Hybrid Cloud
- Multi-Cloud

## Core Components

- Virtual Machines
- Containers
- Kubernetes
- Object Storage
- Managed Databases
- Load Balancers
- CDN

## Engineering Best Practices

- Design for failure.
- Automate infrastructure.
- Monitor everything.
- Secure identities and secrets.
- Optimize for scalability and cost.

Cloud computing forms the backbone of modern software engineering and AI infrastructure.
